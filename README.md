# RPB (Resource Pack Builder)

A resource pack builder that used on a minecraft server that called "Holoright"

## Installation

1. If Node.JS not installed, nstall Node.JS
2. run `npm i -g .`

## Update

1. run `npm remove -g rpb`
2. run `npm i -g .`
